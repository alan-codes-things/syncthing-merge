# Syncthing Merge (`syncthing-merge`)

A cross-platform command-line utility that scans, resolves, and merges Syncthing `.sync-conflict` files.

It integrates directly with your system's global Git configurations to automatically detect and run **whatever visual merge tool you have configured** (Meld, VS Code, KDiff3, Beyond Compare, vimdiff, etc.) exactly as you prefer it.

---

## Key Features

- **Zero-Configuration Tool Integration**: Automatically queries your global Git config (`git config --get merge.tool`) to determine your preferred editor.
- **Compatible with Custom Tools**: If you have a custom program configured via `mergetool.<tool>.cmd`, the script reads that custom execution pattern and dynamically maps the paths.
- **Intelligent Resolving**:
  - Automatically merges non-overlapping edits silently in the background using Git's robust 3-way merge engine.
  - Safely falls back to 2-way comparison if no historical version of the file is preserved in your Syncthing versioning history.
- **Fully Cross-Platform**: Works identically across Windows, macOS, and Linux.

---

## Prerequisites

To run this tool, ensure the following are installed:

1. **Git**: Must be globally accessible from your system PATH.
2. **Your Preferred Merge Tool**: Installed and configured globally in your Git environment (e.g. `git config --global merge.tool vscode` or `meld`).
3. **uv**: Fast Python packaging and tool manager.
